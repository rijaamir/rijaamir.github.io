Accessibility

Ensure accessibility by using descriptive button labels for users using assistive technologies.

Also, all buttons should only direct to one page and clearly state the page they direct to.

Usage

Buttons should be used sparingly. The portfolio site uses them for case studies and for additional content the user may need to navigate to a new page in order to see.

Best Practices

  • Ensure the buttons direct to a working existing page
  • Use hover states for all buttons
